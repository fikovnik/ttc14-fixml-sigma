The SIGMA solutions to the TTC'14 FIXML case study
==================================================

Requirements
------------

- java 7
- maven 3.2.1
- mono-gmcs
- gcc
- g++
- curl

Building
--------

    ./build-all.sh
    ./test-all.sh

The `test-all.sh` will trigger the generation of all the provided test cases and a consecutive compilation for all generated languages.
